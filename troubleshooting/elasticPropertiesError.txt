command
-------
bin/nutch index crawl/crawldb/ crawl/linkdb/ crawl/segments/

Error
-----
Indexer: java.lang.RuntimeException: Missing elastic.cluster and elastic.host. At least one of them should be set in nutch-site.xml

Fix
----
1. Open /opt/apache-nutch-1.13/conf/nutch-site.xml in a textEditor and search for elastic.cluster
2. In value tags put elasticsearch(<value>elasticsearch</value>) and save that file



Reference - https://qbox.io/blog/scraping-the-web-with-nutch-for-elasticsearch
